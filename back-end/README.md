# Agrohelp

## Installation

1. Ouvrez un terminal, puis clonez le projet:

   ```bash
   git clone https://github.com/tech-nova-2025/agrohelp-back.git
   ```

2. Accéder au répertoire du projet:

   ```bash
   cd agrohelp-back
   ```

3. Installez les dépendances:

   ```bash
   npm install
   ```

   ou

   ```bash
   yarn install
   ```

   ou

   ```bash
   pnpm install
   ```

4. Lancer le serveur:
   ```bash
   npm run dev
   ```
